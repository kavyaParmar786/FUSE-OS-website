FAQ

Frequently Asked.

How do I add FUSE OS to my server?

Click the "Add Free" button in the nav or visit the invite link directly. Select your server and grant the required permissions. Setup takes under 30 seconds — no configuration required to start using basic commands.

Is FUSE OS really free?

Yes — completely free, forever. All 150+ commands, all 21 systems, AI chat included. No premium tiers, no credit card, no paywalls. The bot is fully funded and maintained for the community.

What prefix does FUSE OS use?

The default prefix is +. You can change it per-server using +setprefix or from the dashboard.

How does the AI chat work?

FUSE OS uses Groq's llama-3.3-70b model — one of the fastest free AI APIs available. Each user gets their own conversation history that persists across sessions. Use +ai to chat, +imgen to generate images. Admins can configure limits, channels, and system prompts from the dashboard.

How do I set up tickets?

Use +panel1, +panel2, or +panel3 in the channel where you want the ticket panel. Each panel creates a dropdown with different ticket categories. Staff can claim, close, and manage tickets directly from the private channels. Transcripts are auto-saved.

Why isn't the bot responding?

Check that FUSE OS has the correct permissions in your server (Send Messages, Read Messages, Embed Links). Make sure you're using the correct prefix. If the bot appears offline, join our support server and we'll fix it immediately.

How do I report a bug?

Join our Discord support server and post in the bug-reports channel. Include your server ID, the command you used, and what happened vs what you expected. We aim to fix bugs within 24 hours.
