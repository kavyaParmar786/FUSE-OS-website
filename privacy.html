Privacy Policy

How Fuse OS collects, uses, and protects your data.

📅 Last Updated: January 1, 2025 | 📋 Version 2.0 | ✉️ [email protected]

This Privacy Policy describes how Fuse OS ("the Bot", "we", "us", or "our") collects, uses, stores, and shares information when you use our Discord bot service. By adding Fuse OS to your server or interacting with the Bot, you agree to the practices described in this policy.

1. Information We Collect

Fuse OS collects the minimum data necessary to provide its services. We collect the following categories of information:

Discord User Data:

  • Discord User IDs — used to identify users across commands and features
  • Discord Server (Guild) IDs — used to store server-specific configurations
  • Discord Username and Discriminator — used for display in logs and embeds
  • Discord Role IDs — used for autorole and permission management
  • Discord Channel IDs — used for configuration of logging and panel channels

Bot Interaction Data:

  • Command usage logs — timestamps and command names (no message content)
  • Giveaway entries — Discord User IDs linked to giveaway IDs
  • Poll votes — anonymous vote counts and, where enabled, user vote choices
  • Ticket content — message transcripts within ticket channels (server admin-configured)
  • AI conversation history — the last N messages of a user's AI chat session (stored temporarily in memory)
  • Verification codes — temporary codes for YouTube subscription verification

YouTube Integration Data (Optional):

  • YouTube comment scan results — we read comments on a specified video to verify subscription codes
  • YouTube Channel ID — stored in server configuration only; we do not store YouTube viewer data
  • No YouTube account credentials or OAuth tokens are stored by default

2. How We Use Your Data

All data collected by Fuse OS is used exclusively to provide and improve the Bot's features. Specifically:

  • To execute commands and return results to users
  • To persist server configurations between bot restarts
  • To track giveaway entries and select winners
  • To record poll votes and display results
  • To manage ticket creation, assignment, and archival
  • To assign and remove Discord roles automatically
  • To verify YouTube subscription status for Content Fan roles
  • To maintain AI conversation context for the duration of a session
  • To generate leaderboards based on giveaway wins and server activity

We do not use your data for advertising, profiling, resale, or any commercial purpose beyond operating the Bot.

3. Data Storage and Retention

Server configuration data (role IDs, channel IDs, settings) is stored in JSON files on our secure server infrastructure and is retained as long as Fuse OS remains in your Discord server.

Giveaway and poll data is retained for 30 days after the event ends, then permanently deleted.

AI conversation history is stored in-memory only and is cleared when the bot restarts or when a user runs the +aireset command. It is never written to permanent storage.

Verification codes expire after 30 minutes and are removed from storage automatically.

Ticket transcripts are stored only in the Discord channel you configure and are not retained on our servers unless you use the transcript export feature, in which case the export is sent directly to you via Discord DM.

4. Data Sharing and Third Parties

We do not sell, trade, or rent your personal information to third parties. We may share data with the following parties only to the extent necessary to operate the Bot:

  • Anthropic (Claude AI): If you use the AI chat feature, your message content is sent to Anthropic's API to generate a response. Anthropic's own privacy policy governs that data. We do not store AI conversation content after the session ends.
  • Google (YouTube Data API): If you use the YouTube verification feature, we make read-only API calls to check for public comments. No user data is sent to Google beyond what is required for the API request.
  • Discord: All bot interactions occur on Discord's platform. Discord's Privacy Policy governs all data on their platform.

5. Security

We take reasonable technical measures to protect stored data, including:

  • Data stored on servers with restricted access and encrypted disks
  • API keys and credentials stored as environment variables, never in code
  • Regular security reviews of our codebase
  • No storage of sensitive personal information beyond Discord IDs

However, no method of transmission or storage is 100% secure. While we strive to protect your data, we cannot guarantee absolute security.

6. Children's Privacy

Fuse OS is not directed to children under the age of 13. We do not knowingly collect personal information from children under 13. Discord itself requires users to be at least 13 years old. If you believe a child under 13 has provided us with personal information, please contact us immediately.

7. Your Rights and Choices

Depending on your jurisdiction, you may have the right to:

  • Access: Request information about what data we hold about you
  • Deletion: Request deletion of your data by contacting us or kicking the Bot from your server
  • Correction: Request correction of inaccurate data
  • Portability: Request an export of your server's configuration data
  • Objection: Object to specific processing of your data

To exercise any of these rights, please contact us via our support Discord server.

8. Data Deletion

You can delete all data associated with your server by removing Fuse OS from the server. Within 30 days of removal, all server-specific data will be permanently deleted from our systems. Individual users may request deletion of their personal data by contacting support.

9. Changes to This Policy

We may update this Privacy Policy from time to time. We will notify users of significant changes by posting a notice in our support Discord server and updating the "Last Updated" date above. Continued use of the Bot after changes constitutes acceptance of the updated policy.

10. Contact Us

If you have questions, concerns, or requests regarding this Privacy Policy or our data practices, please contact us:

  • Support Server: Join our Discord server and open a support ticket
  • Discord: Direct message the Bot owner via the support server
